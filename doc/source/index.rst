**scikit-rf**（也称为 `skrf`）是一个开源的、采用 BSD 许可的软件包，用于射频/微波工程，并为所有受支持的 Python 编程语言版本（当前为 3.8+）开发和维护。它提供了一个现代的、面向对象的库，既灵活又可扩展。以下文档分为三个部分：叙述性教程、实践示例和参考 API。
