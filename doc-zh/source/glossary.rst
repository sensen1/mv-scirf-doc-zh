.. _glossary:

术语表
=========

.. glossary::

   switch terms（开关项）
      如 Roger Marks 在 [1]_ 中所述，开关项考虑了 8 项（又称 *误差框* ）模型过于简化的事实。两个误差网络会根据哪个端口被激励而轻微变化。这是由于 VNA 内部的开关造成的。

   port impedance（端口阻抗）
      微波网络上给定端口的阻抗。通常
      这与给定传输线模式的特性阻抗相同，
      但也可以任意设置。
      用任何非端口阻抗的阻抗来端接端口
      都会产生反射。

   characteristic impedance（特性阻抗）
      沿传输线传播的单一行波的电压与电流之比。


.. [1] R.B. Marks, "Formulations of the Basic Vector Network Analyzer Error Model including Switch-Terms," in ARFTG Conference Digest-Fall, 50th, 1997, vol. 32, pp. 115-126.

.. [2] https://en.wikipedia.org/wiki/Characteristic_impedance
