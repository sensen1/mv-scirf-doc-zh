**scikit-rf**（也称 `skrf`）是一个开源的、采用 BSD 许可证的射频/微波工程软件包，为 Python 编程语言的所有`受支持版本 <https://www.python.org/downloads/>`_（目前为 3.8+）开发和维护。它提供了一个现代的、面向对象的库，兼具灵活性和可扩展性。本文档分为三个部分：叙述性教程、实用示例和 API 参考。
